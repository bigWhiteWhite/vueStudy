0.git checkout 分支名 //如果在当前分支可省略这步 1.git add . //添加当前分支 2.git cmmit -m ' change ' //准备提交并注释 3.git push //上传分支代码 4.git checkout master //切换到主干 5.git merge origin/分支名 //下载线上分支代码合并到本地主干 6.git push //把本地主干代码上传 7.git上创建分支后，再 git pull 8.进入新创建的分支 git checkout index-ajax 9.可以用git status 来检查是否改动，若没有改动即可 npm run dev 运行项目。
