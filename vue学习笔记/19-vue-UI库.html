Mint UI： https://github.com/ElemeFE/mint-ui 饿了么开源的基于vue的移动端UI组件库 步骤： cnpm i mint-ui -S 然后实现按需打包,ui库里面我可能只是用一些，没必要全部打包: 1.下载cnpm install --save-dev babel-plugin-component 2.修改babel配置，在.babelrc中,other-plugin为其他插件，plugin翻译为插件 "plugins": ["other-plugin", ["component", [ { "libraryName": "mint-ui", "style": true } ]]] 使用： 在main.js上注册成标签 import {Button} from 'mint-ui' Vue.component(Button.name,Button) 有一些可以不用在main上面引入，可以直接在组件引入使用 import { Toast } from 'mint-ui' Toast('你好，世界') Element： 饿了么开源的基于vue的PC端UI组件库 步骤： cnpm i element-ui -S 完整引入： 在 main.js 中写入以下内容 import ElementUI from 'element-ui'; import 'element-ui/lib/theme-chalk/index.css'; Vue.use(ElementUI);//声明使用 按需引入： 借助 babel-plugin-component，我们可以只引入需要的组件，以达到减小项目体积的目的。 首先，安装 babel-plugin-component： 1.下载cnpm install --save-dev babel-plugin-component 然后，将 .babelrc 修改为： "presets": [["es2015", { "modules": false }]], "plugins": [ [ "component", { "libraryName": "element-ui", "styleLibraryName": "theme-chalk" } ] ] 使用： 接下来，如果你只希望引入部分组件，比如 Button 和 Select，那么需要在 main.js 中写入以下内容： import Vue from 'vue'; import { Button, Select } from 'element-ui'; import App from './App.vue'; Vue.component(Button.name, Button); Vue.component(Select.name, Select); /* 或写为 * Vue.use(Button) * Vue.use(Select) */ new Vue({ el: '#app', render: h => h(App) });
